# Cybersecurity Attacks Analysis

## 1. The Question
Do cyberattacks increase during official holidays?

## 2. Hypothesis
I expect the number of attacks to increase during holidays, due to reduced staffing and slower incident response in organizations.

## 3. Data Needed
- Attack date
- Attack type
- Target country
- Targeted sector
- Attack severity

## 4. Data Source
[Kaggle](https://www.kaggle.com/) — searching for public cybersecurity incident datasets.

## 5. Next Steps
- Collect the dataset
- Investigate data quality (missing values, duplicates)
- Clean and prepare the data
- Analyze patterns using Excel/SQL
- Visualize findings and present conclusions
